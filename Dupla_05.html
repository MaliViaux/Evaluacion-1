Debido a estos datos, la prueba seleccionada preliminarmente es el mtodo de ANOVA para muestras independientes. Para realizarla correctamente, es necesario verificar que las condiciones se cumplan correctamente.

  • Observaciones independientes: Debido al formato del estudio, se puede deducir razonablemente que las observaciones son independientes, pues corresponden a distintas personas utilizando un producto para el cabello.
  • La variable medida corresponde a una escala de intervalos iguales: De la misma forma que se mencion en el anlisis preliminar, la medida de microgramos de H2O2 es de intervalos iguales.
  • Las observaciones provienen de una distribucin normal: Para esta condicin, la solucin escogida es realizar un test test de Shapiro-Wilk en cada uno de los grupos.

Pregunta 2

Para la pregunta solicitada, se deben analizar los efectos en las mujeres que usan el placebo en diferentes momentos (3 meses y 6 meses de uso), comparando su percepcin del efecto del producto. Hay que recordar que medida de percepcin est en una escala de Likert, la cual no posee intervalos iguales entre sus valores, por lo que las pruebas tpicas no sern suficientes. Adicionalmente, como se est comparando la percepcin en las mismas mujeres, pero en distintos tiempos totales de uso del producto, las muestras se encuentran correlacionadas. Adicionalmente, no podemos asumir que los datos provienen de una distribucin normal.

datos_filtradosP2 <- datos %>% filter(SEXO == "F") %>% filter(SHAMPOO == "Placebo") %>% select(ID, SEXO, SHAMPOO, PERCEPCION_3, PERCEPCION_6)

head(datos_filtradosP2)

Debido a estos antecedentes, la prueba preliminarmente escogida es la prueba de rangos con signo del Wilcoxon. La prueba normal no sera indicada, pues las muestras no son independientes.

Para esta prueba, se deben cumplir dos condiciones principales:

  • Las observaciones de la muestras deben provenir de una muestra independiente, y deben estar correlacionadas: Como se mencion anteriormente, por la naturaleza del estudio, esta condicin ya est verificada

  • La escala de medida debe ser por lo menos, ordinal. En este caso, como la medida es una escala de Likert, se puede determinar una jerarqua en los puntajes.

Para la prueba se determinan las siguientes hiptesis:

  • HO: Las mujeres sienten una diferencia significativa entre los 3 y 6 meses de utilizar un shampoo placebo
  • HA: Las mujeres no sienten una diferencia significativa entre los 3 y 6 meses de utilizar un shampoo placebo
alfa <- 0.05

# Hacer la prueba de wilcoxon de signos
prueba <- wilcox.test(datos_filtradosP2$PERCEPCION_3, datos_filtradosP2$PERCEPCION_6,paired = TRUE, alternative = "two.sided", conf.level = 1 - alfa)
print(prueba)
## 
##  Wilcoxon signed rank test with continuity correction
## 
## data:  datos_filtradosP2$PERCEPCION_3 and datos_filtradosP2$PERCEPCION_6
## V = 280884, p-value = 0.6001
## alternative hypothesis: true location shift is not equal to 0

A partir de la prueba seleccionada, con un nivel de significancia de 0.05, y un p-valor mucho ms alto que el alfa seleccionado, no se logra rechazar la hiptesis nula, concluyendo con un 95% de confianza que no existe suficiente evidencia para demostrar que las mujeres sienten una diferencia significativa entre los tres y seis meses de utilizar el producto placebo.
